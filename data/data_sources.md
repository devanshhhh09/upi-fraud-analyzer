# UPI Fraud Analyzer — Data Sources

## Category A: Official Structured Reports

### NCRB Annual Crime Report 2022
- URL: https://ncrb.gov.in/en/Crime-in-India-2022
- Format: PDF
- Contains: State-wise cyber crime counts, UPI fraud category breakdowns
- Access: Public download
- Status: To be downloaded

### RBI Annual Report 2023-24
- URL: https://www.rbi.org.in/Scripts/AnnualReportPublications.aspx
- Format: PDF
- Contains: Digital payment fraud statistics, UPI-specific data
- Access: Public download
- Status: To be downloaded

### CERT-In Advisories
- URL: https://www.cert-in.org.in/s2cMainServlet?pageid=PUBVLNOTES01
- Format: HTML
- Contains: Specific fraud campaign descriptions and IOCs
- Access: Public
- Status: To be scraped

## Category B: Semi-structured Public Sources

### cybercrime.gov.in awareness section
- URL: https://www.cybercrime.gov.in
- Format: HTML
- Contains: Fraud awareness articles with real case descriptions
- Access: Public
- Status: To be scraped

### PIB Press Releases
- URL: https://pib.gov.in
- Format: HTML
- Contains: Cyber fraud arrest reports, MO descriptions
- Access: Public
- Status: To be scraped

### NewsAPI
- URL: https://newsapi.org
- Format: JSON API
- Contains: News articles about UPI fraud
- Access: Free tier (100 requests/day)
- Status: API key required
